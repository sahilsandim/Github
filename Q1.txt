What is Version control system?
Version control is a tool to track a changes in files,documents,and websites
also it helps in coordinating work with multiple developers.
-->This tool is used to keep the directories,project organised.
-->Involved in large software projects.
-->It helps to store large files and helps to track changes in the repositories
